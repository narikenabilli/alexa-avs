Help

If you are out of survey

Command Description 
start Predix Survey
 Start survey from the beginning. 
ask Predix Survey to continue
 Continue survey if it was started and not finished. Useful when you excidently left survey. 
tell Predix Survey that [you answer]
 Give your answer if survey was started and not finished. Useful when you excidently left survey. 
ask Predix Survey for help
 Provides help about how to start survey and answer questions.
